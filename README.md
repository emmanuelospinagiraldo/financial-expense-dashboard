# financial-expense-dashboard
Financial Expense Dashboard built with Node.js, Express, PostgreSQL and vanilla JavaScript.   This application allows real-time expense tracking, categorization by cost centers, monthly filtering, Excel report generation and interactive financial insights with charts.  Designed as a practical solution for business treasury management.
# 💼 Financial Expense Dashboard

A full-stack financial management system for tracking and analyzing business expenses.

## 🚀 Features

- Register daily expenses
- Categorize by cost center and type
- Monthly filtering
- Excel report generation
- Real-time dashboard
- Financial insights and analytics
- Interactive charts

## 🛠️ Tech Stack

- Node.js
- Express
- PostgreSQL
- Vanilla JavaScript
- Chart.js
